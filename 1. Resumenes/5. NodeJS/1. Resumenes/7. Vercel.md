# VERCEL
Vercel es una plataforma en la nube que permite a los desarrolladores desplegar, gestionar y escalar sus apps y sitios web.

## 1. Creamos un fichero

```
Nombre del fichero:

vercel.json
```

## 2. Dentro del fichero vercel.json debemos agregar?

```
{
    "version": 2,
    "name": "electrosa",
    "builds": [
        {
            "src": "index.js",
            "use": "@vercel/node"
        }
    ],
    "routes": [
        {
            "src": "/(.*)",
            "dest": "/index.js"
        }
    ]
}
```
